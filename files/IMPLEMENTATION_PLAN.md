## STRUCTURE TEMPLATE

Every page must include:

1. **Lesson title and objectives**
2. **Dialogue or reading passage**
3. **Vocabulary table** (using `.vocab-table`) - minimum 10 words
4. **Grammar explanation boxes** (using `.grammar-box` variants) - 2-3 per page
5. **Drill sections** (using `.drill-section`) - 3-4 per page
6. **Cultural note** (using `.cultural-note`)
7. **Summary/review section**

---

## QUALITY STANDARDS

### Content Requirements
- File size: 6KB minimum (target 15-20KB for comprehensive lessons)
- Vocabulary table: 10-15 new words per lesson
- Grammar sections: 2-3 per page
- Drill exercises: 3-4 per page
- Cultural context: 1 section per page

### Design Standards
- NO inline styles (zero tolerance)
- Use only classes defined in style.css
- Consistent heading hierarchy (h1 > h2 > h3)
- Proper semantic HTML structure

### Language Standards
- Main text: Turkish
- Technical terms: English
- Russian examples: Cyrillic with transliteration
- Verb forms: Feminine (for female learner persona)
