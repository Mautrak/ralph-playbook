Genitive Case: Rusca'nin Iyelik ve Miktar Hali

Accusative Case'i tamamladik, simdi Rusca'nin en cok kullanilan Case'lerinden biri olan Genitive Case'e (Roditel'niy Padej - Родительный падеж) geciyoruz. Genitive, Turkce'deki -in, -nin, -un, -nun eklerine benzer ama cok daha genis kullanim alanlarina sahip.

Genitive Case Nedir?

Genitive Case soyle sorulara cevap verir:

  • Кого? (Kavo?) - Kimin? (canli)
  • Чего? (Chevo?) - Neyin? (cansiz)

Temel anlami: Sahiplik, ait olma, bir seyin parcasi olma. Turkce'de "Alinin kitabi", "sunun rengine" gibi.

Bolum 1: Temel Donusum Alistirmalari

Alistirma 1.1: Isimleri Genitive'e Cevir

Asagidaki isimleri Genitive formuna donustur:

  1. книга (kitap) → книги
  2. друг (arkadas) → друга
  3. сестра (kizkardes) → сестры
  4. брат (erkek kardes) → брата
  5. вода (su) → воды
  6. вино (sarap) → вина
  7. девушка (kiz) → девушки
  8. парень (erkek) → парня
  9. мама (anne) → мамы
  10. море (deniz) → моря
  11. чай (cay) → чая
  12. любовь (ask) → любви

Alistirma 1.2: Cinsiyet ve Sonek Eslestirme

Asagidaki ismin cinsiyetini ve Genitive sonekini belirle:

  1. стол (masa) → Masculine, -a: стола
  2. работа (is) → Feminine, -ы: работы
  3. окно (pencere) → Neuter, -а: окна
  4. музей (muze) → Masculine, -я: музея
  5. неделя (hafta) → Feminine, -и: недели
  6. здание (bina) → Neuter, -я: здания
  7. ночь (gece) → Feminine, -и: ночи
  8. день (gun) → Masculine, -я: дня

Bolum 2: Preposition + Genitive Alistirmalari

Alistirma 2.1: Dogru Preposition Sec

Asagidaki cumlelerde dogru preposition'u sec (из, от, до, без, для, у, около, после):

  1. ___ Турции (Turkiye'den) → из Турции
  2. ___ тебя (senin icin) → для тебя
  3. ___ утра (sabaha kadar) → до утра
  4. ___ друга (arkadastan - bir sey almak) → от друга
  5. ___ меня (bende) → у меня
  6. ___ сахара (sekersiz) → без сахара
  7. ___ работы (isten sonra) → после работы
  8. ___ дома (evin yakininda) → около дома

Alistirma 2.2: Cumle Tamamla

Bosluklari dogru Genitive formlarla doldur:

  1. Я из ___. (Россия - Rusya) → Я из России.
  2. Это для ___. (ты - sen) → Это для тебя.
  3. Я работаю до ___. (вечер - aksam) → Я работаю до вечера.
  4. Кофе без ___. (молоко - sut) → Кофе без молока.
  5. После ___ пойдём гулять? (работа - is) → После работы пойдём гулять?
  6. У ___ есть время? (ты - sen) → У тебя есть время?
  7. Я живу около ___. (центр - merkez) → Я живу около центра.
  8. Это подарок от ___. (он - o erkek) → Это подарок от него.

Bolum 3: "У меня есть" Yapisi

Alistirma 3.1: Sahiplik Cumleleri Kur

Verilen bilgiyle "У ... есть ..." yapisinda cumle yaz:

  1. Ben + araba → У меня есть машина.
  2. Sen + erkek arkadas → У тебя есть парень.
  3. O (kadin) + kedi → У неё есть кошка.
  4. Biz + zaman → У нас есть время.
  5. Onlar + para → У них есть деньги.
  6. O (erkek) + kiz arkadas → У него есть девушка.

Alistirma 3.2: Soru Formuna Cevir

Asagidaki cumleleri soru formuna cevir:

  1. У тебя есть машина. → У тебя есть машина? (Arabam var mi?)
  2. У неё есть брат. → У неё есть брат? (Erkek kardesi var mi?)
  3. У вас есть время. → У вас есть время? (Zamaniniz var mi?)
  4. У него есть работа. → У него есть работа? (Isi var mi?)

Bolum 4: Turkce'den Rusca'ya Ceviri

Alistirma 4.1: Cumle Cevirisi

Asagidaki Turkce cumleleri Rusca'ya cevir:

  1. "Arkadasin evi" → дом друга
  2. "Bende telefon var." → У меня есть телефон.
  3. "Sensiz gitmek istemiyorum." → Я не хочу идти без тебя.
  4. "Bu senin icin." → Это для тебя.
  5. "Turkiye'denim." → Я из Турции.
  6. "Isten sonra bulusuruz." → Встретимся после работы.
  7. "Sabaha kadar bekledim." → Я ждала до утра.
  8. "Evin yakininda bir kafe var." → Около дома есть кафе.

Alistirma 4.2: Mini Diyalog Tamamla

Bosluklari dogru Genitive formlarla doldur:

A:
Ты откуда? (Nerelisin?)
B:
Я из ___. (Istanbul) Стамбула
A:
У ___ есть время сегодня вечером? (sen) тебя
B:
Да! Хочешь встретиться после ___? (is) работы
A:
Отлично! Кафе около ___ в 7? (metro) метро (degismez)

Bolum 5: Accusative vs Genitive Karistirma Alistirmasi

Ne Zaman Accusative, Ne Zaman Genitive?

Bu iki Case bazen karistirilir. Temel ayrimi yap:

DurumCaseOrnek
Dogrudan nesne (gormek, sevmek)AccusativeЯ вижу девушку.
SahiplikGenitiveЭто книга девушки.
в/на + hareketAccusativeИду в ресторан.
из/от + kaynakGenitiveИду из ресторана.
Miktar ifadesiGenitiveСтакан воды.

Alistirma 5.1: Dogru Case'i Sec

Parantezdeki kelimeyi dogru Case'e cevir:

  1. Я вижу ___. (девушка) → девушку (Accusative - dogrudan nesne)
  2. Это книга ___. (девушка) → девушки (Genitive - sahiplik)
  3. Я иду в ___. (ресторан) → ресторан (Accusative - hareket)
  4. Я иду из ___. (ресторан) → ресторана (Genitive - из + Gen)
  5. Хочу стакан ___. (вода) → воды (Genitive - miktar)
  6. Я люблю ___. (она) → её (Accusative - dogrudan nesne)
  7. Это для ___. (она) → неё (Genitive - для + Gen)
  8. Я жду ___. (друг) → друга (Accusative canli = Genitive formuyla ayni!)

Rus Kulturu: Misafirperverlik ve Sahiplik

Ruslarda «У меня есть...» yapisi sadece maddi seyleri degil, iliskileri de ifade eder:

  • У меня есть друзья в Москве. - Moskova'da arkadaslarim var.
  • У меня есть парень/девушка. - Erkek arkadasim/kiz arkadasim var.

Bir Rus'a «У тебя есть девушка/парень?» diye sormak cok dogal ve yaygindir. Bu soru "iliski durumunu" sormak icin standart yontemdir.

Flort ipucu: «У тебя есть парень?» (Erkek arkadasin var mi?) - Bu cumlede тебя Genitive formundadir!

Bolum 6: Flort Baglaminda Genitive

Alistirma 6.1: Romantik Cumleler

Asagidaki Turkce cumleleri Rusca'ya cevir (Genitive kullanarak):

  1. "Erkek arkadasin var mi?" → У тебя есть парень?
  2. "Sensiz yapamiyorum." → Я не могу без тебя.
  3. "Bu sana ozel bir hediye." → Это подарок для тебя.
  4. "Numaran var mi?" → У тебя есть номер? / Дай мне свой номер.
  5. "Sabaha kadar seninle olmak istiyorum." → Я хочу быть с тобой до утра.
  6. "Isten sonra bulusalim mi?" → Встретимся после работы?
  7. "Gozlerinin rengi ne guzel!" → Какой красивый цвет глаз! (gozlerinin - глаз Genitive plural)

Alistirma 6.2: Ilk Tanisma Diyalogu

Bosluklari tamamla:

A:
Привет! Ты откуда?
B:
Я ___ Украины. А ты? из
A:
Я ___ Турции. У ___ есть планы на вечер? из, тебя
B:
Пока нет. А что?
A:
Пойдём куда-нибудь ___ работы? после
B:
С удовольствием! ___ меня есть любимое кафе около ___. У, метро/центра/дома

Pratik Oneri: Her gun en az 5 cumle kur: "Bende ... var" (У меня есть...), "Sensiz..." (Без тебя...), "Senin icin..." (Для тебя...). Bu yapilar flort konusmalarinda cok isine yarayacak!

Bu Derste Ogrendiklerimiz

  • Genitive Case nedir: Sahiplik, miktar, "-nin/-in" anlami tasiyan Case
  • Soru kelimeleri: Кого? (Kimin?), Чего? (Neyin?)
  • Masculine sonekler: -а / -я (друг → друга)
  • Feminine sonekler: -ы / -и (книга → книги)
  • Neuter sonekler: -а / -я (вино → вина)
  • Temel preposition'lar: из, от, до, без, для, у, около, после
  • Sahiplik yapisi: У меня есть... (Bende ... var)
  • Zamir Genitive: меня, тебя, его, её, нас, вас, их
