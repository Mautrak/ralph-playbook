Dative Case: Rusca'nin Dolayli Nesne Hali

Genitive Case'i tamamladik, simdi ucuncu onemli Case olan Dative Case'e (Datel'niy Padej - Дательный падеж) geciyoruz. Dative, Turkce'deki -e, -a, -ye, -ya (yonelme hali) eklerine benzer. "Kime?", "Neye?" sorularina cevap verir.

Dative Case Nedir?

Dative Case soyle sorulara cevap verir:

  • Кому? (Kamu?) - Kime? (canli)
  • Чему? (Chemu?) - Neye? (cansiz)

Temel anlami: Dolayli nesne - bir eylemin KIME yoneldigi. Turkce'de "Aliye verdim", "anneme soyledim" gibi.

Neden Onemli? Dative Case, flort konusmalarinda cok kullanilir: "Sana bir sey soyleyecegim" (Я хочу тебе что-то сказать), "Seni begendim" (Ты мне нравишься). Bu yapilar olmadan romantik iletisim kurmak zor!

Bolum 1: Temel Donusum Alistirmalari

Alistirma 1.1: Isimleri Dative'e Cevir

Asagidaki isimleri Dative formuna donustur:

  1. книга (kitap) → книге
  2. друг (arkadas) → другу
  3. сестра (kizkardes) → сестре
  4. брат (erkek kardes) → брату
  5. мама (anne) → маме
  6. папа (baba) → папе
  7. девушка (kiz) → девушке
  8. парень (erkek) → парню
  9. учитель (ogretmen) → учителю
  10. ночь (gece) → ночи
  11. море (deniz) → морю
  12. окно (pencere) → окну

Alistirma 1.2: Cinsiyet ve Sonek Eslestirme

Asagidaki ismin cinsiyetini ve Dative sonekini belirle:

  1. стол (masa) → Masculine, -у: столу
  2. работа (is) → Feminine, -е: работе
  3. музей (muze) → Masculine, -ю: музею
  4. неделя (hafta) → Feminine, -е: неделе
  5. здание (bina) → Neuter, -ю: зданию
  6. любовь (ask) → Feminine (-ь), -и: любви
  7. врач (doktor) → Masculine, -у: врачу
  8. Катя (Katya - isim) → Feminine, -е: Кате

Bolum 2: Zamir Dative Alistirmalari

Alistirma 2.1: Dogru Zamiri Sec

Bosluklari dogru Dative zamirle doldur:

  1. Дай ___ (ben) книгу. → Дай мне книгу.
  2. Скажи ___ (o - erkek) правду. → Скажи ему правду.
  3. Позвони ___ (sen)! → Позвони мне! (Beni ara!)
  4. Помоги ___ (o - kadin)! → Помоги ей!
  5. Покажи ___ (biz) город. → Покажи нам город.
  6. Напиши ___ (siz)! → Напишите нам! / Напиши мне!
  7. Ответь ___ (onlar)! → Ответь им!
  8. Верь ___ (ben)! → Верь мне!

Alistirma 2.2: "Нравиться" Cumleleri

Asagidaki Turkce cumleleri "нравиться" yapisini kullanarak Rusca'ya cevir:

  1. "Senden hoslaniyorum." → Ты мне нравишься.
  2. "Bu muzik hosuma gidiyor." → Эта музыка мне нравится.
  3. "Ondan hoslaniyorum." (erkek) → Он мне нравится.
  4. "Ondan hoslaniyorum." (kadin) → Она мне нравится.
  5. "Bu film hosuna gidiyor mu?" → Тебе нравится этот фильм?
  6. "Onun gozleri hosuma gidiyor." → Мне нравятся его/её глаза.
  7. "Senden cok hoslaniyorum!" → Ты мне очень нравишься!
  8. "Rusya hosuna gidiyor mu?" → Тебе нравится Россия?

Bolum 3: Preposition + Dative Alistirmalari

Alistirma 3.1: "К" Preposition'u

Bosluklari "к" + Dative ile tamamla:

  1. Иду ___ (sen). → Иду к тебе.
  2. Приходи ___ (ben)! → Приходи ко мне!
  3. Идём ___ (arkadas). → Идём к другу.
  4. Еду ___ (anne). → Еду к маме.
  5. ___ (aksam) буду дома. → К вечеру буду дома.
  6. Подойди ___ (o - kadin). → Подойди к ней.
  7. Иду ___ (doktor). → Иду к врачу.
  8. Поедем ___ (onlar)! → Поедем к ним!

Alistirma 3.2: "По" Preposition'u

Bosluklari "по" + Dative ile tamamla:

  1. Скучаю ___ (sen). → Скучаю по тебе.
  2. Гуляю ___ (sokak). → Гуляю по улице.
  3. Говорю ___ (telefon). → Говорю по телефону.
  4. Скучаю ___ (o - kadin). → Скучаю по ней.
  5. ___ (benim fikrim) это хорошо. → По моему мнению это хорошо. / По-моему это хорошо.
  6. Иду ___ (park). → Иду по парку.

Bolum 4: Yas Ifadeleri

Alistirma 4.1: Yas Sorusu ve Cevaplari

Asagidaki cumleleri Rusca'ya cevir:

  1. "Kac yasindasin?" → Сколько тебе лет?
  2. "25 yasindayim." → Мне двадцать пять лет.
  3. "O 21 yasinda." (kadin) → Ей двадцать один год.
  4. "O 18 yasinda." (erkek) → Ему восемнадцать лет.
  5. "Biz ayni yastayiz." → Нам одинаковое количество лет. / Мы одного возраста.
  6. "Cok genc gorunuyorsun!" → Ты выглядишь очень молодо!

Bolum 5: Turkce'den Rusca'ya Ceviri

Alistirma 5.1: Temel Cumleler

Asagidaki Turkce cumleleri Rusca'ya cevir:

  1. "Bana yardim et!" → Помоги мне!
  2. "Ona (kadin) bir sey soyleyecegim." → Я скажу ей кое-что.
  3. "Bana gercegi soyle." → Скажи мне правду.
  4. "Sana numarami vereyim." → Дам тебе свой номер.
  5. "Beni ara!" → Позвони мне!
  6. "Bana yaz!" → Напиши мне!
  7. "Ona inanma!" → Не верь ему! / Не верь ей!
  8. "Bize sehri goster!" → Покажи нам город!

Alistirma 5.2: Flort Cumleleri

Asagidaki flort cumlelerini Rusca'ya cevir:

  1. "Senden hoslaniyorum." → Ты мне нравишься.
  2. "Seni ozluyorum." → Скучаю по тебе.
  3. "Sana geliyorum." → Иду к тебе.
  4. "Bana gel!" → Приходи ко мне!
  5. "Numarani ver bana." → Дай мне свой номер.
  6. "Seninle olmak istiyorum." → Хочу быть с тобой.
  7. "Gulumsemen hosuma gidiyor." → Мне нравится твоя улыбка.
  8. "Bana bir seyler icir." → Купи мне выпить.

Alistirma 7.1: Dogru Case'i Sec

Parantezdeki kelimeyi dogru Case'e cevir:

  1. Я вижу ___. (девушка) → девушку (Accusative - dogrudan nesne)
  2. Я дала ___ книгу. (друг) → другу (Dative - dolayli nesne)
  3. Это машина ___. (папа) → папы (Genitive - sahiplik)
  4. Помоги ___! (она) → ей (Dative - помогать fiili)
  5. У меня нет ___. (время) → времени (Genitive - нет ile)
  6. Ты ___ нравишься. (я) → мне (Dative - нравиться fiili)
  7. Скучаю по ___. (ты) → тебе (Dative - по ile)
  8. Я люблю ___. (она) → её (Accusative - dogrudan nesne)

Rus Kulturu: Dative ve Duygusal Ifadeler

Rusca'da duygusal durumlar genellikle Dative ile ifade edilir. Bu yapiya dikkat et:

  • Мне хорошо. - Kendimi iyi hissediyorum.
  • Мне плохо. - Kendimi kotu hissediyorum.
  • Мне холодно. - Usuyorum. (lit: Bana soguk)
  • Мне жарко. - Sicakliyorum. (lit: Bana sicak)
  • Мне скучно. - Sikiliyorum.
  • Мне интересно. - Ilgimi cekiyor. / Merak ediyorum.

Flort ipucu: «Мне с тобой хорошо» (Seninle kendimi iyi hissediyorum) romantik bir ifadedir.

Bu Derste Ogrendiklerimiz

  • Dative Case nedir: Dolayli nesne, "kime/neye" sorusuna cevap verir
  • Soru kelimeleri: Кому? (Kime?), Чему? (Neye?)
  • Masculine/Neuter sonekler: -у / -ю (друг → другу)
  • Feminine sonekler: -е (cogu) veya -и (sadece -ь ile bitenler)
  • Zamir Dative: мне, тебе, ему, ей, нам, вам, им
  • Temel fiiller: давать, говорить, помогать, звонить, писать, нравиться
  • Preposition'lar: к (-e dogru), по (boyunca, ozlemek)
  • "Нравиться" yapisi: Ты мне нравишься (Senden hoslaniyorum)
  • Yas ifadesi: Мне 25 лет (25 yasindayim)
  • Duygusal ifadeler: Мне хорошо/плохо/холодно/жарко
