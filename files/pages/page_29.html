Bolum 23: Genitive Case - Kapsamli Pratik ve Alistirmalar

Sayfa 29 / 40

Genitive Case: Tum Konularda Yogun Pratik

Son bes derste Genitive Case'in butun yonlerini ogrendik: temel sonekler, sahiplik, olumsuzluk, sayilar ve iliski konusmalari. Simdi kapsamli pratik yaparak butun bu bilgileri pekistirecegiz. Bu ders Genitive Case'in "final sinavi" gibi - kendini test et!

Hatirlatma: Genitive Case, Rusca'nin en cok kullanilan ve en onemli case'lerinden biri. Bu alistirmalari duzenli tekrar etmek, sokakta konusurken Genitive'i dogal sekilde kullanmani saglayacak.

Bolum 1: Temel Genitive Donusumleri

Alistirma 1.1: Isim Donusumu - Tekil

Asagidaki isimleri Genitive formuna donustur:

  1. книга (kitap) → книги
  2. друг (arkadas - erkek) → друга
  3. студентка (ogrenci - kadin) → студентки
  4. вино (sarap) → вина
  5. мама (anne) → мамы
  6. папа (baba) → папы
  7. кофе (kahve) → кофе (degismez)
  8. парень (erkek arkadas) → парня
  9. девушка (kiz) → девушки
  10. ночь (gece) → ночи
  11. учитель (ogretmen) → учителя
  12. море (deniz) → моря
  13. вода (su) → воды
  14. муж (koca) → мужа
  15. жена (kari) → жены

Alistirma 1.2: Cogul Genitive Donusumu

Asagidaki cogul isimleri Genitive formuna donustur:

  1. книги (kitaplar) → книг
  2. друзья (arkadaslar) → друзей
  3. столы (masalar) → столов
  4. окна (pencereler) → окон
  5. девушки (kizlar) → девушек
  6. дети (cocuklar) → детей
  7. года / годы (yillar) → лет (duzensiz!)
  8. люди (insanlar) → людей
  9. рубли (ruble) → рублей
  10. минуты (dakikalar) → минут

Bolum 2: Sahiplik (Possession) Alistirmalari

Sahiplik Yapilari - Hatirlatma

Rusca'da sahiplik icin iki yapi kullanilir:

  • Isim + Genitive: книга друга (arkadasin kitabi)
  • У + Genitive + есть: У меня есть книга (kitabim var)

Soru: Чей? Чья? Чьё? Чьи? (Kimin?)

Alistirma 2.1: "Kimin?" Sorulari

Asagidaki sorulari Rusca'ya cevir ve cevapla:

  1. "Bu kimin kitabi?" (arkadas) → Чья это книга? — Это книга друга.
  2. "Bu kimin arabasi?" (baba) → Чья это машина? — Это машина папы.
  3. "Bu kimin telefonu?" (kizkardes) → Чей это телефон? — Это телефон сестры.
  4. "Bunlar kimin anahtarlari?" (anne) → Чьи это ключи? — Это ключи мамы.
  5. "Bu kimin evi?" (ogretmen) → Чей это дом? — Это дом учителя.
  6. "Bu kimin kiz arkadasi?" (Dima) → Чья это девушка? — Это девушка Димы.

Alistirma 2.2: Sahiplik Cumleleri

Asagidaki cumleleri Rusca'ya cevir:

  1. "Arkadasimin evi buyuk." → Дом друга большой.
  2. "Annemin yemegi cok lezzetli." → Еда мамы очень вкусная.
  3. "Erkek arkadasimin arabasi kirmizi." → Машина моего парня красная.
  4. "Rusya'nin baskenti Moskova." → Столица России — Москва.
  5. "Istanbul'un nufusu buyuk." → Население Стамбула большое.
  6. "Kizin gozu mavi." → Глаза девушки голубые.

Bolum 3: Olumsuzluk (Negation) Alistirmalari

Нет + Genitive - Hatirlatma

Rusca'da "yok" demek icin нет + Genitive kullanilir:

  • У меня есть книга. (Kitabim var.) - Nominative
  • У меня нет книги. (Kitabim yok.) - Genitive

Gecmis zaman: не было + Gen. | Gelecek zaman: не будет + Gen.

Alistirma 3.1: Var → Yok Donusumu

"Есть" cumlesini "нет" cumlesine cevir:

  1. У меня есть время. → У меня нет времени.
  2. У него есть деньги. → У него нет денег.
  3. У неё есть парень. → У неё нет парня.
  4. У нас есть машина. → У нас нет машины.
  5. У тебя есть работа. → У тебя нет работы.
  6. В комнате есть окно. → В комнате нет окна.
  7. У них есть дети. → У них нет детей.
  8. У вас есть вопросы. → У вас нет вопросов.

Alistirma 3.2: Turkce'den Rusca'ya - Olumsuzluk

Asagidaki olumsuz cumleleri Rusca'ya cevir:

  1. "Param yok." → У меня нет денег.
  2. "Erkek arkadasim yok." → У меня нет парня.
  3. "Zamanim yok." → У меня нет времени.
  4. "Burada su yok." → Здесь нет воды.
  5. "Sorularim yok." → У меня нет вопросов.
  6. "Planlarimiz yok." → У нас нет планов.
  7. "Problemi yok." → У него нет проблем. / У неё нет проблем.
  8. "Burada restoran yok." → Здесь нет ресторана.

Alistirma 3.3: Gecmis ve Gelecek Zaman Olumsuzluk

Cumleleri gecmis veya gelecek zamana cevir:

  1. У меня нет денег. (gecmis) → У меня не было денег.
  2. У неё нет парня. (gelecek) → У неё не будет парня.
  3. Здесь нет воды. (gecmis) → Здесь не было воды.
  4. У нас нет времени. (gelecek) → У нас не будет времени.
  5. У него нет работы. (gecmis) → У него не было работы.

Asagidaki yaslari Rusca'ya cevir:

Asagidaki fiyatlari Rusca'ya cevir:

  2. "Kahve 150 ruble." → Кофе стоит сто пятьдесят рублей.
  3. "Bu kac para?" → Сколько это стоит?

Alistirma 5.1: Preposition + Genitive Cumleleri

Bosluklari doldurun:

  1. Я из ___. (Turkiye) → Я из Турции.
  2. Это подарок для ___. (sen) → Это подарок для тебя.
  3. Кофе без ___. (seker) → Кофе без сахара.
  4. После ___ пойдём гулять. (is) → После работы пойдём гулять.
  5. До ___ далеко? (metro) → До метро далеко?
  6. Я получила письмо от ___. (anne) → Я получила письмо от мамы.
  7. Около ___ есть кафе. (ev) → Около дома есть кафе.
  8. Из-за ___ мы опоздали. (yagmur) → Из-за дождя мы опоздали.
  9. Вместо ___ пришла её сестра. (o - kadin) → Вместо неё пришла её сестра.
  10. У ___ красивая квартира. (arkadas) → У друга красивая квартира.

Alistirma 5.2: Turkce'den Rusca'ya - Prepositionlar

Asagidaki cumleleri Rusca'ya cevir:

  1. "Turkiye'den geliyorum." → Я из Турции.
  2. "Seker olmadan iciyorum." → Я пью без сахара.
  3. "Senin icin getirdim." → Я принесла для тебя.
  4. "Isten sonra bulusalim." → Давай встретимся после работы.
  5. "Aksama kadar bos musun?" → Ты свободна до вечера?
  6. "Yagmur yuzunden eve kaldik." → Из-за дождя мы остались дома.
  7. "Evin yakininda park var." → Около дома есть парк.
  8. "Benim yerime git." → Иди вместо меня.

Bolum 6: Iliski ve Sozyal Durumlar

Alistirma 6.1: Iliski Cumleleri

Asagidaki cumleleri Rusca'ya cevir:

  1. "Erkek arkadasim var." → У меня есть парень.
  2. "Erkek arkadasim yok." → У меня нет парня.
  3. "Uc yildir birlikte." → Мы вместе три года.
  4. "Eski erkek arkadasimdan ayrildim." → Я рассталась с бывшим парнем.
  5. "Onun kiz arkadasi cok guzel." → Его девушка очень красивая.
  6. "Serbestim." (kadin) → Я свободна.
  7. "Evliyim." (kadin) → Я замужем.
  8. "Acik iliskideyiz." → Мы в открытых отношениях.

Alistirma 6.2: Iliski Soruları

Asagidaki sorulari Rusca'ya cevir:

  1. "Erkek arkadasin var mi?" → У тебя есть парень?
  2. "Evli misin?" (kadina) → Ты замужем?
  3. "Serbest misin?" (kadina) → Ты свободна?
  4. "Ne kadardir birliktesiniz?" → Сколько вы вместе?
  5. "Eskisi var mi?" → У него есть бывшая? / У неё есть бывший?
  6. "Ciddi mi?" → Это серьёзно?

Bolum 9: Cumle Duzeltme

Alistirma 9.1: Hatalari Duzelt

Asagidaki cumlelerde Genitive hatalari var. Duzelt:

  1. У меня нет время. → У меня нет времени. (время → времени)
  2. Два рубль. → Два рубля. (рубль → рубля)
  3. Без сахар. → Без сахара. (сахар → сахара)
  4. Это книга друг. → Это книга друга. (друг → друга)
  5. У неё нет парень. → У неё нет парня. (парень → парня)
  6. Пять год. → Пять лет. (год → лет)
  7. После работа. → После работы. (работа → работы)
  8. Много деньги. → Много денег. (деньги → денег)
  9. Для ты. → Для тебя. (ты → тебя)
  10. От она. → От неё. (она → неё)

Bolum 11: Karisik Egzersizler

Alistirma 11.1: Hizli Donusum Testi

30 saniye icinde Genitive formunu yaz:

  1. я → меня
  2. книга → книги
  3. друг → друга
  4. время → времени
  5. деньги → денег
  6. год (5 ile) → лет
  7. она (preposition ile) → неё
  8. работа → работы
  9. парень → парня
  10. вода → воды

Alistirma 11.2: Cumle Yazma Yarismasi

Verilen kelimelerle Genitive kullanan bir cumle yaz:

  1. нет + время → У меня нет времени.
  2. без + ты → Мне плохо без тебя.
  3. много + работа → У меня много работы.
  4. после + ужин → После ужина пойдём гулять.
  5. для + она → Это подарок для неё.
  6. из + Россия → Он из России.
  7. 25 + год → Мне двадцать пять лет.
  8. машина + друг → Это машина друга.

Alistirma 11.3: Karisik Preposition Testi

Dogru preposition'u sec ve cumleyi tamamla:

  1. "Turkiye'den geldim." (из / от) → Я приехала из Турции. (из = icinden, yerden)
  2. "Anneden mektup aldim." (из / от) → Я получила письмо от мамы. (от = kisiden)
  3. "Isim yuzunden yorgunum." (от / из-за) → Я устала из-за работы. (из-за = yuzunden)
  4. "Isten yoruldum." (от / из-за) → Я устала от работы. (от = -den, dogrudan sebep)
  5. "Aksama kadar bosun?" (до / после) → Ты свободна до вечера?

Genitive Case - Pratik Sirlar

Sokakta konusurken Genitive'i dogal kullanmak icin bu ipuclarina dikkat et:

  • En sik kullanim: "У меня нет..." (... yok) - bu yapiyi gunluk hayatta surekli kullanacaksin.
  • Sayilarla dikkat: 2, 3, 4 ile Genitive tekil; 5+ ile Genitive cogul. "Лет" kelimesi her zaman cogul!
  • Preposition'lar ezberle: из, без, для, от, до, после - bunlar hep Genitive alir.
  • Zamirlerde н-: Preposition'dan sonra его → него, её → неё, их → них.
  • Iliski konusmasi: "У меня есть парень" / "У меня нет парня" - bu ikili asla unutma!

Gunluk pratik: Her gun en az 5 cumle kur: "Param yok", "3 kahve istiyorum", "Turkiye'denim", vs.
