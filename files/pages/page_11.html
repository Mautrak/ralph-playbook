Sahiplik - "Bende Var" Yapisi (Possession in Russian)

Bu derste Ruscanin en farkli ve en onemli yapilarindan birini ogreneceksin: sahiplik ifadesi. Turkce "Arabam var" veya Ingilizce "I have a car" gibi dogrudan "sahip olmak" fiili yerine, Rusca tamamen farkli bir yaklasim kullaniyor. Bu yapi У меня есть kalibini kullanir - kelime kelime "Benim yaninda var" anlamina gelir. Rusca dusunce sistemini anlamak icin bu yapiyi kavramak kritik!

Temel Sahiplik Yapisi: У + Genitif + есть

Rusca'da "X'in Y'si var" demek icin su formul kullanilir:

UnsurAciklamaOrnek
У"-da/-de" edati (yaninda)U
Genitif HalSahip olan kisi (Genitive Case)меня (benim)
есть"var" (existence)yest'
Yalin HalSahip olunan sey (Nominative)машина (araba)
У меня есть машина. U meNYA yest' maSHIna. Arabam var. (Kelime kelime: "Benim yaninda var araba")

Felsefe Farki: Turkce "Arabam var" derken "araba" sana ait bir sey. Rusca'da ise "senin yaninada/uzerinde bir araba mevcut" gibi dusunuluyor. Bu, Rus dilinin "sahiplik" kavramina daha az vurgu yapmasindan kaynaklaniyor - seyler sana ait degil, sadece senin yaninada!

Kisisel Zamirlerin Genitif Hali (Genitive Pronouns)

Sahiplik Icin Gerekli Zamir Formlari

У edati her zaman Genitif Hal (Genitive Case) gerektirir. Iste kisisel zamirlerin genitif formları:

KisiYalin HalGenitif HalOkunusuSahiplik Yapisi
ben я меня meNYA У меня (bende)
sen (samimi) ты тебя teRYA У тебя (sende)
o (erkek/nesne) он/оно него neVO У него (onda-erkek)
o (kadin) она неё neYO У неё (onda-kadin)
biz мы нас nas У нас (bizde)
siz вы вас vas У вас (sizde)
onlar они них nikh У них (onlarda)

Dikkat: 3. sahis zamirleri (o, onlar) У edatindan sonra н- harfi alir: его → него, её → неё, их → них

Sik Yapilan Hata: "У его есть..." YANLIS! Dogru kullanim: "У него есть...". Edat+zamir kombinasyonlarinda 3. sahis hep н- alir!

Olumsuz Sahiplik: У меня нет + Genitif

Yokluk Ifadesi - нет Kullanimi

"...yok" demek icin есть yerine нет kullanilir. ONEMLI: нет'ten sonra isim Genitif Hal'e girer!

Olumlu (есть + Nominatif)Olumsuz (нет + Genitif)
У меня есть машина. (araba-nom) У меня нет машины. (araba-gen)
У неё есть брат. (erkek kardes-nom) У неё нет брата. (erkek kardes-gen)
У них есть дети. (cocuklar-nom) У них нет детей. (cocuklar-gen)
У меня есть время. (zaman-nom) У меня нет времени. (zaman-gen)

Genitif Hal Ornekleri:

  • Erkek isimler: unsuz → +а/я (брат → брата, учитель → учителя)
  • Kadin isimler: -а/-я → -ы/-и (машина → машины, неделя → недели)
  • Nesne isimler: -о/-е → -а/-я (окно → окна, море → моря)
— У тебя есть сигарета? — Нет, у меня нет сигарет. — U teRYA yest' sigaRYEta? — Nyet, u meNYA nyet sigaRYET. — Sigaran var mi? — Hayir, sigaram yok.

Soru Yapisi (Question Formation)

"...var mi?" Sorusu

Sahiplik sorulari cok basit: ayni cumleyi soru tonlamasiyla soyle veya basina А ekle:

SoruOkunusuTurkce
У тебя есть машина? U teRYA yest' maSHIna? Araban var mi?
У вас есть паспорт? U vas yest' PASpart? Pasaportunuz var mi?
У него есть девушка? U neVO yest' DYEvushka? Kiz arkadasi var mi?
А у тебя есть брат? A u teRYA yest' brat? Peki senin erkek kardesin var mi?

Kisa Cevaplar:

  • Да, есть. (Evet, var.)
  • Нет, нет. veya Нет, у меня нет. (Hayir, yok.)

Есть'i Ne Zaman Atlayabiliriz?

Есть vs. Есть'siz Kullanim

Bazi durumlarda есть atlanabilir. Kural: Varolus mu tanimlama mi?

DurumЕсть KullanimiOrnek
VAROLUS - "...var mi?" anlaminda есть KULLANILIR У тебя есть машина? (Araban var mi?)
TANIMLAMA - "...nasil?" anlaminda есть ATLANIR У меня (—) красивая машина. (Guzel bir arabam var.)
Yeni bilgi, ilk kez bahsediliyor есть KULLANILIR У меня есть сестра. (Kiz kardesim var - ilk kez soyluyorum)
Bilinen seyin ozelligi есть ATLANIR У меня (—) хорошая сестра. (Iyi bir kiz kardesim var - taniyorsun)
— У тебя есть квартира? — Да, у меня большая квартира в центре. — U teRYA yest' kvarTIra? — Da, u meNYA balSHAya kvarTIra v TSENtre. — Dairen var mi? — Evet, merkezde buyuk bir dairem var.

Dikkat: Ilk cumle varolus soruyor (есть gerekli), ikinci cumle tanimlama yapiyor (есть atlanabilir).

Elif:

Elif:

Diyalogdan Sahiplik Kaliplari

  • У тебя есть сигарета? - Soru: varolus (есть kullaniliyor)
  • У меня нет - Olumsuz kisa cevap
  • У тебя есть телефон? - Soru: varolus
  • У меня есть - Olumlu kisa cevap
  • У тебя красивая сумка! - Tanimlama: есть ATLANMIS (guzel cantan var = cantan guzel)

Diyalogdan Olumsuz Ornekler

  • У меня нет сестры - сестра (nom) → сестры (gen)
  • У меня нет парня - парень (nom) → парня (gen)
  • Нет времени - время (nom) → времени (gen)

Not: У твоей сестры - iyelik zamiri (твоей) ve isim (сестры) ikisi de Genitif!

Rusya'da "Var mi?" Sorusu

Ruslar У тебя есть...? sorusunu bazen daha dogrudan bir sekilde sorar. Ornegin, bir partide tanismak icin У тебя есть парень/девушка? (Erkek/kiz arkadasin var mi?) sorusu ilk birkac dakika icinde gelebilir. Bu, Turk kulturunde "cok acik" sayilabilir ama Rusya'da gayet normaldir. Ruslar genellikle dolaylamaktan kacinir ve merak ettiklerini dogrudan sorarlar.

Maddi Durum ve Sahiplik

Sovyet doneminden kalan bir kulturel ozellik: Ruslar genellikle У меня нет денег (Param yok) demekten cekinmezler. Turkiye'de "param yok" demek utanilacak bir sey olabilirken, Rusya'da bu cok normaldir. Hatta bazi Ruslar maddi durumlarini abartili derecede kotu gosterme egilimindedir - bu "nazar degmesin" dusuncesinden ziyade, muhafazakar bir tutum olarak goruluyor. У меня всё есть (Her seyim var) demek ise biraz gosterisci algilanabilir.

Genitif Zamir Secimi

Dogru genitif zamiri secin (меня, тебя, него, неё, нас, вас, них):

  1. У ___________ есть машина. (bende) - меня
  2. У ___________ есть брат? (sende) - тебя
  3. У ___________ нет времени. (onda - erkek) - него
  4. У ___________ красивая квартира. (onda - kadin) - неё
  5. У ___________ есть вопросы? (sizde) - вас
  6. У ___________ нет денег. (onlarda) - них

Olumlu → Olumsuz Donusum

Cumleleri olumsuz hale getirin (genitif kullanarak):

  1. У меня есть машина. - У меня нет машины.
  2. У неё есть брат. - У неё нет брата.
  3. У них есть дети. - У них нет детей.
  4. У тебя есть работа. - У тебя нет работы.
  5. У нас есть время. - У нас нет времени.
  6. У него есть девушка. - У него нет девушки.

Turkce → Rusca Ceviri

Asagidaki cumleleri Rusca'ya cevirin:

  1. Arabam var. - У меня есть машина.
  2. Erkek kardesin var mi? - У тебя есть брат?
  3. Param yok. - У меня нет денег.
  4. Onun (kadin) kocasi var. - У неё есть муж.
  5. Zamanin var mi? - У тебя есть время?
  6. Sorunumuz yok. - У нас нет проблемы / проблем.

Есть mi, Есть'siz mi?

Cumlelerde есть kullanilmali mi, atlanmali mi? Belirtin:

  1. У тебя ___ машина? (Araban var mi? - ilk kez soruyorum) - есть (varolus sorusu)
  2. У меня ___ хорошая машина. (Iyi bir arabam var - tanimlama) - atlanir (tanimlama)
  3. У вас ___ паспорт? (Pasaportunuz var mi? - resmi) - есть (varolus sorusu)
  4. У неё ___ красивые глаза. (Guzel gozleri var - tanimlama) - atlanir (tanimlama)

Isimlerle Sahiplik

Verilen bilgiyle cumle kurun:

  1. Katya / kopek / var - У Кати есть собака.
  2. Ivan / araba / yok - У Ивана нет машины.
  3. annem / kiz kardes / var - У мамы есть сестра.
  4. arkadasim (erkek) / is / yok - У друга нет работы.

Soru - Cevap Eslestirme

Sorulara uygun cevap verin:

  1. У тебя есть телефон? (Evet, var) - Да, у меня есть. veya Да, есть.
  2. У вас есть вопросы? (Hayir, yok) - Нет, у нас нет. veya Нет, нет.
  3. У него есть сестра? (Evet, iki kiz kardesi var) - Да, у него есть две сестры.
  4. У тебя есть деньги? (Hayir, param yok) - Нет, у меня нет денег.

Bu Derste Ogrendiklerimiz

  • Temel Yapi: У + Genitif + есть + Nominatif = "...in ...si var"
  • Genitif Zamirler: меня, тебя, него, неё, нас, вас, них
  • 3. Sahis Kurali: У edatindan sonra н- eklenir: него, неё, них
  • Olumsuz: У меня нет + Genitif - isim genitif hale girer!
  • Soru: Ayni cumle, soru tonlamasiyla - У тебя есть...?
  • Есть vs. Есть'siz: Varolus soruluyorsa есть kullan, tanimlama yapiliyorsa atlayabilirsin
  • Isimlerle: У Кати есть... - isim de genitife girer
  • Kisa Cevaplar: Да, есть. / Нет, нет.
  • Gunluk Kaliplar: У меня нет времени (Zamanim yok), У меня нет денег (Param yok)
